# Get 100DaysOfCode info from twitter

## Getting started

1. Set up dotenv

   ```config
   API_KEY=...
   API_KEY_SECRET=...
   BEARER_TOKEN=...
   ```

1. Run code

   ```bash
   # First time
   python3 -m venv venv
   python3 -m pip install --upgrade pip 
   python3 -m pip install -r requirements.txt

   # Later
   . ./venv/bin/activate
   python3 ./main.py
   ```
